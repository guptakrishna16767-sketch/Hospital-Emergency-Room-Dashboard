# 🏥 Hospital Emergency Room Dashboard

An end-to-end **Healthcare Data Analytics Project** built using **Python** and **Power BI** to analyze Emergency Room (ER) operations and transform raw healthcare data into meaningful business insights.

---

## 📖 Project Overview

This project demonstrates the complete data analytics workflow—from data cleaning and preprocessing in **Python** to creating an interactive dashboard in **Power BI**.

The dashboard provides valuable insights into hospital emergency room performance, helping stakeholders monitor patient flow, waiting times, admissions, referrals, and patient satisfaction through interactive visualizations.

---

## 🎯 Objectives

- Analyze Emergency Room performance
- Monitor patient waiting times
- Evaluate patient satisfaction
- Track admission and referral trends
- Understand patient demographics
- Identify peak patient arrival hours
- Support data-driven healthcare decisions

## 🔄 Project Workflow

1. Collected hospital emergency room dataset.
2. Cleaned and preprocessed data using **Python**.
3. Transformed data using **Power Query**.
4. Built relationships and created calculated measures using **DAX**.
5. Designed interactive dashboards in **Power BI**.
6. Generated business insights to support decision-making.
